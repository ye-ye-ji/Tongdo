# Tongdo Fantasia

통도환타지아는 대한민국 양산시에 위치한 놀이공원으로 아쿠아환타지아 워터파크가 함께 입지하고 있습니다.<br>
이러한 브랜드 특성을 반영하기 위해서 대표컬러를 파란색으로 선정했으며 해당색상이 주는 활력과 안정감을 고려하여 깔끔하고 직선적인 서체를 사용했습니다.
컨텐츠를 카드UI로 구성하고 상단에 검색영역을 추가하여 사용자의 접근성과 시인성을 높였습니다.
